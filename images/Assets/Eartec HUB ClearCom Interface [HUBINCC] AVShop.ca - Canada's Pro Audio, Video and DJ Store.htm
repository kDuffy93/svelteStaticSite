About: Eartec HUB ClearCom Interface

The Hub Clear-Com Interface  - Wireless to wired interface

The HUB interface is a simple, affordable way to add full duplex wireless headsets to popular wired Clear-com intercoms.  The interface connects the HUB (sold separetly) to the wall jack or rack-mounted power console. Signal level adjustments create a seamless audio link to any combination of up to 6 transceivers. Chose any combination of wireless headsets to create a completely customized system.

  • Add six wireless at a time
  • Roaming users communicate with partyline
  • All communicate in full duplex

Roaming users can communicate with wired headsets hands free, simultaneously, and without interconnecting wires.
